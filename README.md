# 🚀 Arduino LED Resistor Calculator

A fast, mobile-friendly tool for hobbyists and engineers to calculate the exact resistor needed for LED circuits. Built with Tailwind CSS and hosted on GitHub Pages.

## ✨ Features
- **Smart Presets:** Common LED forward voltages (Red, Blue, Green, etc.) built-in.
- **Wattage Safety Check:** Automatically warns you if you need a high-power resistor.
- **Reference Chart:** High-contrast resistor color code table included.
- **Responsive Design:** Works perfectly on phones, tablets, and desktops.

## 🛠️ How it Works
The calculator uses Ohm's Law: 
$R = \frac{V_{supply} - V_{forward}}{I_{current}}$
